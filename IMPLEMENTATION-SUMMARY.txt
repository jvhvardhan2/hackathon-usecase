================================================================================
USAGE INSTRUCTIONS
================================================================================

Quick Start:
1. Follow QUICKSTART.md for 10-minute setup
2. Or follow DEPLOYMENT-GUIDE.md for step-by-step
3. Or read SETUP.md for comprehensive guide

Deploy Infrastructure:
$ cd terraform/environments/dev
$ terraform init
$ terraform apply

Deploy Applications:
$ kubectl apply -f k8s/base/

Trigger CI/CD:
$ git push origin main
(GitHub Actions handles build, push, deploy)

================================================================================
MONITORING & OPERATIONS
================================================================================

View Logs:
$ gcloud logging read "resource.type=k8s_container AND resource.labels.namespace_name=healthcare"

View Pods:
$ kubectl get pods -n healthcare

View Metrics:
$ kubectl top pods -n healthcare

Access Dashboard:
https://console.cloud.google.com/monitoring/dashboards

Port Forward for Testing:
$ kubectl port-forward svc/patient-service 3000:3000 -n healthcare
